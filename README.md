# Телеграм-бот с несколькими ИИ-моделями. А также с подписками и лимитами + админ-панель.

## Возможности
- Выбор ИИ - Yandex/DeepSeek(заглушки)
- Лимиты запросов для пользователей
- Подписки (Free / Pro)
- Админ-панель
- Готовая архитектура для подключения существующих API

## Архитектура
- Bot.py - тут вся логика бота
- users.py - пользователи, лимиты и подписки
- models.py - работа с ИИ(заглушки)
- keyboards.py - кнопки и меню
- config.py - конфигурация(не хранится в репозитории)

## Установка
   bash
   pip install pytelegrambotapi
   
## Дополнительно
Создайте у себя config.py, в переменные BOT_TOKEN и ADMIN_IDS вставьте токен вашего бота, а также ID своего(их) аккаунта(ов), чтобы можно было получить доступ к админ-панели.
Запустите введя команду - python Bot.py

## Админ-команды
/admin - сама админ-панель, показывает количество пользователей.
/give - выдаёт подписку пользователю. Формат: /give <тут должен быть id пользователя> limit
